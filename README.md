# Структура
```markdown
charybdis-4x6-wellum/
├── keyboards/
│   └── bastardkb/
│       └── charybdis/
│           └── 4x6/
│               └── v2/
│                   └── splinky_2/            ← файлы для компиляции
│                       ├── config.h            ← основная конфигурация
│                       ├── info.json           ← информация о клавиатуре
│                       ├── mcuconf.h           ← конфигурация микроконтроллера
│                       ├── readme.md           ← документация
│                       ├── rules.mk            ← правила компиляции
│                       └── keymaps/
│                           └── wellum/          ← раскладка Wellum
│                               ├── keymap.c       ← оcпределение раскладки клавиатуры и логики работы слоев
│                               ├── rules.mk       ← настройка параметров компиляции для раскладки
│                               └── config.h       ← пользовательские настройки, специфичные для раскладки
└── .github/
    └── workflows/
        └── manual_build.yml            ← критически важный файл для вашей CI/CD системы
```
